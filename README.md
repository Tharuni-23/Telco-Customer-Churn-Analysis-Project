# Telco Customer Churn Analysis

## Project Overview

This repository contains my work completed as part of the Business Analysis Internship at SaiKet Systems. The project focuses on analyzing the Telco Customer Churn dataset using Microsoft Excel to understand customer behavior, identify churn patterns, and derive meaningful business insights through data exploration and visualization techniques.

## Dataset Information

* Dataset Name: Telco Customer Churn Dataset
* Total Records: 7,043 Customer Records
* Total Attributes: 21 Variables

## Project Objectives

* Explore and understand the dataset structure.
* Analyze customer demographics and service usage patterns.
* Identify customer churn trends and retention metrics.
* Create visualizations and summary reports.
* Derive actionable business insights from the data.

## Tools Used

* Microsoft Excel
* Data Sorting and Filtering
* Data Visualization Techniques
* Basic Business Analysis Methods

## Project Tasks

* Task 1: Data Overview & Simple Sorting
* Task 2: Churn Count Visualization
* Additional tasks will be added as the internship progresses.
