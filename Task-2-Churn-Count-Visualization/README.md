# Task 2: Churn Count Visualization

## Objective
To visualize the distribution of churned and retained customers using Microsoft Excel charts.

## Dataset Information
- Dataset Name: Telco Customer Churn Dataset
- Total Records: 7,043 Customer Records
- Total Attributes: 21 Variables

## Activities Performed
- Created a customer churn summary table.
- Generated a clustered column chart to compare churned and retained customers.
- Added chart elements such as title, data labels, and axis labels.
- Derived business insights from churn and retention percentages.

## Key Insights
- Retained Customers: 73.46%
- Churned Customers: 26.54%
- Approximately 73% of customers were retained, while 27% of customers churned.

## Chart Details
- Chart Title: Customer Churn Distribution
- X-Axis Label: Customer Status
- Y-Axis Label: Number of Customers

- Microsoft Excel
- Data Visualization
- Basic Business Analysis Techniques
