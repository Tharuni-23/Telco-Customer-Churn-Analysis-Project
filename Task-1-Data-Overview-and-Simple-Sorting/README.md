# Task 1: Data Overview & Simple Sorting

## Objective
To explore the Telco Customer Churn dataset and perform basic sorting and filtering operations using Microsoft Excel.

## Dataset Information
- Dataset Name: Telco Customer Churn Dataset
- Total Records: 7,043 Customer Records
- Total Attributes: 21 Variables

## Activities Performed
- Explored the dataset structure and customer information.
- Sorted customers based on tenure.
- Filtered churned customers for further analysis.
- Created summary tables and derived key business insights.

## Key Insights
- Total Customers: 7,043
- Churned Customers: 1,869 (26.54%)
- Retained Customers: 5,174 (73.46%)
- Tenure Range: 0–72 Months
- New Joinees: 11

- Basic Business Analysis Techniques
