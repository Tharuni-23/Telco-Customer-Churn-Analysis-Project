# Task 4: Churn Rates by Gender Analysis

## Objective

To compare customer churn rates across genders using Pivot Tables and Clustered Bar Charts.

## Dataset Information

* Dataset Name: Telco Customer Churn Dataset
* Total Records: 7,043 Customer Records
* Attributes Analyzed: Gender and Churn

## Tasks Performed

* Created a Pivot Table to calculate churn counts for each gender.
* Compared churned and non-churned customer counts across male and female customers.
* Created a Clustered Bar Chart using Pivot Table data.
* Applied distinct colors to represent churned and non-churned categories.
* Added descriptive chart titles, axis labels, and key insights.

## Pivot Table Summary

| Gender | Non-Churned (No) | Churned (Yes) | Total Customers |
| ------ | ---------------: | ------------: | --------------: |
| Female |            2,549 |           939 |           3,488 |
| Male   |            2,625 |           930 |           3,555 |
| Total  |            5,174 |         1,869 |           7,043 |

## Key Insights

* Female customers account for 49.53% of the total customer base.
* Male customers account for 50.47% of the total customer base.
* Female churn rate is 26.92%.
* Male churn rate is 26.16%.
* Non-churned customers are higher than churned customers for both genders.
* Customer churn is almost equally distributed across both genders, indicating that gender has minimal influence on churn behavior.
